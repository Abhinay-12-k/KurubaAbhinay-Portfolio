# Kuruba Abhinay — Premium Developer Portfolio

![Portfolio Preview Banner](https://img.shields.io/badge/Status-Live%20on%20Render-success?style=for-the-badge&logo=render)

A world-class, highly-interactive developer portfolio meticulously crafted to showcase full-stack engineering expertise, AI integrations, problem-solving skills, and key achievements. The design is inspired by top-tier tech product landings, utilizing a rich, light-themed aesthetic with modern geometric typography and beautiful glassmorphism interfaces.

🚀 **Live Demo:** [https://kurubaabhinay-portfolio.onrender.com/](https://kurubaabhinay-portfolio.onrender.com/)

---

## ✨ Key Features

- **Expressive Hero Section:** Features a stunning 3D interactive orbital animation alongside premium typography and dynamic mesh gradients.
- **Bento Grid Layouts:** Employs a modern, component-driven bento grid approach to elegantly display skills, status, and system architecture.
- **Immersive Micro-Interactions:** Smooth Framer Motion animations across all sections—scroll reveals, hover lifts, glowing borders, and counting stats.
- **Glassmorphism Design System:** Consistent translucent UI cards that hover above complex background gradients and subtile noise overlay textures.
- **Responsive Architecture:** Fully optimized for all screens with a seamless mobile menu and floating pill navigation.

---

## 🛠️ Technology Stack

This project was built with modern web development best practices:

- **Framework:** React.js + Vite (for ultra-fast HMR and optimized production builds)
- **Styling:** Tailwind CSS + Custom CSS Variables 
- **Animations:** Framer Motion
- **3D Graphics:** React Three Fiber (`@react-three/fiber` & `@react-three/drei`)
- **Icons:** Lucide React
- **Typography:** Clash Display, Plus Jakarta Sans, DM Sans, JetBrains Mono, Syne
- **Deployment:** Render (Static Site)

## 🚀 Running Locally

To run this repository locally on your machine:

**1. Clone the repository**
```bash
git clone https://github.com/Abhinay-12-k/KurubaAbhinay-Portfolio.git
cd KurubaAbhinay-Portfolio
```

**2. Install dependencies**
```bash
npm install
```

**3. Start the development server**
```bash
npm run dev
```

**4. Build for production**
```bash
npm run build
```
